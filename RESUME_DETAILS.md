# AI Job Application Tracker

## 📌 Project Overview
**Name:** AI-powered Job Application Tracker
**Role:** Full Stack Developer
**Tech Stack:** 
- **Frontend:** React 19, Tailwind CSS v4, Vite
- **Backend:** Node.js, Express.js, JWT Authentication
- **Database:** MongoDB
- **Tools/Libraries:** PDF-Parse (Resume Analysis), Axios, Multer (File Handling)

---

## 💻 Bullet Points for "My Projects" Section
*Pick 3-4 bullet points that best highlight your strengths:*

- **Full-Stack Development & Architecture:** Designed and developed an end-to-end Job Application Tracker using the MERN stack (MongoDB, Express, React, Node.js), enabling users to seamlessly manage their applications, track statuses, and analyze job compatibility.
- **AI-Driven Resume Analysis:** Engineered a heuristic ATS-style parsing engine on the backend using `pdf-parse` and text-processing algorithms, offering users a real-time keyword match score against job descriptions.
- **Skill Gap Computation:** Implemented logic to extract core focus terms from job descriptions and cross-reference them with parsed resumes, automatically generating actionable feedback on missing skills and targeted resume revisions.
- **Secure Architecture & File Handling:** Built a secure authentication system with JWT encryption and integrated `multer` to handle and validate secure PDF document uploads.
- **Modern & Responsive UI/UX:** Crafted a highly responsive and visually engaging frontend with React 19 and Tailwind CSS v4, utilizing a customized design system and micro-animations to enhance user interaction experience.

## 🎯 Short Summary / Elevator Pitch
*"I built a full-stack AI Job Application Tracker using the MERN stack that functions like a personal ATS. Users can upload their resumes in PDF format, paste a job description, and the application's backend engine parses the text to instantly provide a keyword match score. It also generates personalized suggestions and a skill gap analysis, helping job seekers tailor their applications effectively. The frontend was built with React 19 and Tailwind CSS v4 for a highly polished, responsive user experience."*
